# E-commerce – Backend

Backend API developed with Node.js, Express and TypeScript.

## Tech stack
- Node.js
- Express
- TypeScript
- JWT authentication
- File-based storage (JSON)

## Features
- User registration & authentication (JWT)
- Product management (admin only)
- Shopping cart management
- Wishlist management

## Getting started
```bash
npm install
npm run dev
```
## Scripts

- npm run dev – development
- npm run build – build
- npm start – production
